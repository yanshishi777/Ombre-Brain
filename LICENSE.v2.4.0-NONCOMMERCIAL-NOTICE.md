# Ombre Brain v2.4.0 Noncommercial Source Notice

This document records the intended v2.4.0 source-public distribution policy for Ombre Brain.
It is a source-available notice, not legal advice, and it does not delete or rewrite the
historical `LICENSE` file already present in this repository. Before publishing v2.4.0 under
binding new terms, the project owner should review the final license choice.

## Allowed Intent

Ombre Brain v2.4.0 source is intended to remain visible and usable for:

- personal learning
- research
- private experimentation
- noncommercial self-hosting
- noncommercial forks that preserve the original attribution and this notice

## Restricted Commercial Use

The following use requires prior written permission from the project owner:

- commercial hosting
- paid resale
- renamed resale
- SaaS resale
- selling modified builds
- packaging Ombre Brain v2.4.0 as a paid hosted memory service
- removing attribution while distributing modified copies

## Why This Notice Exists

Ombre Brain is designed as AI-native collaborative memory infrastructure, not as a
generic commodity database wrapper. Public source helps learning and inspection, but
the project owner does not intend to grant permission for unapproved paid resale or
renamed commercial redistribution of v2.4.0 work.
